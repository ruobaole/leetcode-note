## leetcode shuati
- 第一阶段
    - 按专题刷
    - 每个专题完成: easy -- 60%, medium -- 50%
- 第二阶段
    - 巩固知识脉络和手感
    - easy: 5min; medium 20min;
    - 每天刷easy保持手感; 辅助几道medium巩固知识;
- 第三阶段
    - 方向1) 刷hard接触高级data structure & algorithm --> 线段树
    - 方向2) 培养应试感觉 --> 参加每周leetcode比赛

- 小技巧
    - 通过最优时间复杂度找思路
    - 如何知道最优时间复杂度？看 N 范围：
        - 和 10^8 比较, e.g:
        - N = 10^6 --> N^2 > 10^8 --> O(N)
        - N = 10^3 --> N^2 = 10^6 < 10^8 --> O(N^2) or O(NlogN)
        - N <= 20 --> O(2^N)
